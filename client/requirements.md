## Packages
recharts | For visualizing expense data (bar charts, pie charts)
date-fns | For robust date formatting and manipulation
framer-motion | For smooth page transitions and micro-interactions
clsx | For conditional class names (often used with tailwind-merge)
tailwind-merge | For merging tailwind classes

## Notes
Integration with Object Storage is required for receipt uploads.
The app uses an AI analysis endpoint (/api/receipts/analyze) to auto-fill form data.
